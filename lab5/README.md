# lab5 许可认证软件

本实验使用 Windows Socket API 实现一个控制台版许可认证系统，包含服务器和客户端两部分。

## 功能

- 服务器端可通过控制台购买许可证，返回 10 位序列号。
- 客户端首次运行时输入序列号，后续运行可自动读取本地配置。
- 客户端向服务器请求授权，服务器按许可证人数上限进行放行或拒绝。
- 客户端运行期间会周期性发送心跳，服务器根据心跳清理异常退出的会话。
- 服务器状态会写入本地文件，重启后可恢复已授权会话信息。

## 编译

建议使用 Visual Studio 的 C++ 控制台项目，将 [code.cpp](code.cpp) 加入工程，并链接 `Ws2_32.lib`。

也可以直接使用命令行编译，例如：

```bat
cl /EHsc code.cpp Ws2_32.lib
```

## 运行

先启动服务器：

```bat
code.exe server 5050 license_state.db
```

服务器控制台命令：

- `buy <user> <password> <type>`: 购买许可证，例如 `buy admin 123456 10`
- `status`: 查看当前许可证和会话状态
- `quit`: 退出服务器

再启动客户端：

```bat
code.exe client 127.0.0.1 5050 client_license.txt 30
```

客户端会在首次运行时提示输入序列号，并将序列号、会话令牌和客户端标识保存到本地文件。
